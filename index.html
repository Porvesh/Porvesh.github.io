Enflied Engine

Enflied is a 2D multiplatform Multi Threaded C++ based Game Engine with a User Interface with extensive support for input methods and customization.

The engine is coded in C++, hosts Lua scripting, and utilizes GLM, LuaBridge, Rapidjson, SDL2, BOX2D, ImGui.

Lua Integration Classes and Functions

Actor Class

  • GetName() - Retrieve the actor's name.
  • GetID() - Fetch the unique identifier of the actor.
  • GetComponentByKey(key) - Get a component of the actor by its key.
  • GetComponent(type) - Get a component based on its type.
  • GetComponents() - Retrieve all components attached to the actor.
  • Find(name) - Static function to find an actor by name.
  • FindAll(name) - Static function to find all actors by name.
  • AddComponent(component) - Add a component to the actor from Lua.
  • RemoveComponent(component) - Remove a component from the actor from Lua.
  • Instantiate() - Static function to create a new actor instance.
  • Destroy() - Static function to destroy an actor.

Application Namespace

  • Quit() - Exits the application.
  • Sleep(milliseconds) - Delays the application for a specified number of milliseconds.
  • GetFrame() - Returns the current frame count since the application started.
  • OpenURL(url) - Opens a URL in the default web browser.

Input Class

  • GetKey(keyCode) - Returns true if the specified key is currently being pressed.
  • GetKeyDown(keyCode) - Returns true if the specified key was pressed down in the current frame.
  • GetKeyUp(keyCode) - Returns true if the specified key was released in the current frame.
  • GetMousePosition() - Returns the current position of the mouse cursor as a 'vec2'.
  • GetMouseButton(button) - Returns true if the specified mouse button is currently being pressed.
  • GetMouseButtonDown(button) - Returns true if the specified mouse button was pressed down in the current frame.
  • GetMouseButtonUp(button) - Returns true if the specified mouse button was released in the current frame.
  • GetMouseScrollDelta() - Returns the amount the mouse wheel has been scrolled in the current frame.

TextDB Class

  • Draw(text, x, y, scale, color) - Draws text at the specified position with given scale and color.

AudioDB Class

  • Play(soundID) - Plays the sound effect associated with the given sound ID.
  • Halt(soundID) - Halts playback of the sound effect associated with the given sound ID.
  • SetVolume(soundID, volume) - Sets the volume for the sound effect associated with the given sound ID.

ImageDB Class

  • DrawUI(imageID, x, y, width, height) - Draws a UI image at the specified coordinates.
  • DrawUIEx(imageID, x, y, width, height, rotation, color) - Draws a UI image with additional parameters like rotation and color.
  • Draw(imageID, x, y) - Draws an image at the specified screen coordinates.
  • DrawEx(imageID, x, y, scaleX, scaleY, rotation, color) - Draws an image with scaling, rotation, and color adjustments.
  • DrawPixel(x, y, color) - Draws a single pixel at the specified screen coordinates.

CameraBounds Class

  • SetPosition(x, y) - Sets the camera's position in the game world.
  • GetPositionX() - Returns the current X-coordinate of the camera's position.
  • GetPositionY() - Returns the current Y-coordinate of the camera's position.
  • SetZoom(zoomLevel) - Sets the zoom level of the camera.
  • GetZoom() - Retrieves the current zoom level of the camera.

Scene Class

  • Load(sceneName) - Loads a new scene, transitioning from the current scene.
  • GetCurrent() - Returns the name of the current active scene.
  • DontDestroy(object) - Marks an object to not be destroyed when loading a new scene.

Rigidbody Class

  • GetPosition() - Returns the current position.
  • GetRotation() - Returns the current rotation angle.
  • AddForce(forceVec) - Applies a force vector to the Rigidbody.
  • SetVelocity(velocityVec) - Sets the velocity vector of the Rigidbody.
  • SetPosition(x, y) - Sets the position of the Rigidbody.
  • SetRotation(angle) - Sets the rotation of the Rigidbody.
  • GetVelocity() - Returns the current velocity vector.
  • SetUpDirection(directionVec) - Sets the 'up' direction vector.
  • GetUpDirection() - Gets the current 'up' direction vector.

Collision and b2Vec2 Classes

  • other - The other object involved in the collision.
  • point - The point at which the collision occurred.
  • relative_velocity - The velocity relative to the other object at the point of impact.
  • normal - The collision normal vector.

Physics and HitResult Classes

  • Raycast(startPoint, endPoint) - Performs a raycast from start to end point, returning a HitResult.
  • RaycastAll(startPoint, endPoint) - Performs a raycast that returns all hits along the ray's path.

EventBus Class

  • Publish(eventType, eventArgs) - Publishes an event to all subscribed listeners.
  • Subscribe(eventType, listener) - Subscribes a listener function to an event type.
  • Unsubscribe(eventType, listener) - Unsubscribes a listener from an event type.

Vector2 Class

  • Normalize() - Normalizes the vector so that its length is 1.
  • Length() - Returns the length (magnitude) of the vector.
  • Distance(v1, v2) - Calculates the distance between two vectors.
  • Dot(v1, v2) - Computes the dot product of two vectors.
  • __add - Allows adding two Vector2 objects with the '+' operator.
  • __sub - Allows subtracting two Vector2 objects with the '-' operator.
  • __mul - Allows multiplying a Vector2 object with a scalar or another Vector2 with the '*' operator.

GameManager Functions

  • AddGame(game) - Adds a game to the list.
  • DeleteGame(game) - Removes a game from the list.
  • ListGames() - Displays all games in the list.
  • FindGame(game_name) - Checks if a game exists in the list.
  • SortGames() - Sorts the list of games alphabetically.
  • RandomizeGames() - Shuffles the list of games.
  • LoadGamesFromFile(filename) - Loads games from a specified file.
  • SaveGamesToFile(filename) - Saves the current list of games to a file.
  • CountGames() - Returns the number of games in the list.
  • RemoveAllGames() - Clears the list of games.
  • MergeLists(otherList) - Compares and merges another list into the current list.
